📈 R2 Stocks Widget

Fast, embeddable stock quote widget with real-time data and observability

Default Theme - AAPL
Dark Theme - MSFT
Custom Theme - GOOGL

Widget Controls

🚀 Quick Start

<!-- Add container to your page -->
<div id="stocks-widget"></div>

<!-- Include the widget script -->
<script src="stocks.bundle.js"></script>

<!-- Initialize the widget -->
<script>
  StocksSnapshot.init({
    containerId: 'stocks-widget',
    symbol: 'AAPL',
    apiKey: 'YOUR_API_KEY',
    refreshInterval: 60000,
    theme: {
      primaryColor: '#007bff',
      backgroundColor: '#ffffff',
      textColor: '#333333'
    }
  });
</script>
